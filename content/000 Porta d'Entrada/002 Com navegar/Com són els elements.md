# Com són els elements del Metalaboratori?

Aquest entorn està format per diferents tipus d’elements.  
Cada element té una funció dins el sistema i ajuda a organitzar el coneixement.

Abans de continuar, recorda que sempre pots tornar enrere:

↩️ [[Com funciona]]

---

## Tipus d’elements que trobaràs

### 🔹 Nodes conceptuals

Són les peces bàsiques del sistema.

Representen conceptes clau de la recerca i actuen com a punts de connexió entre idees.

Exemples:  
→ [[Cognició distribuïda]]  
→ [[Arquitectura pedagògica]]  
→ [[Co-associació]]

Cada node conceptual et permet:

- entendre una idea central
    
- veure com es relaciona amb altres conceptes
    
- accedir a desenvolupaments teòrics i aplicacions pràctiques
    

---

### 🔹 Dominis del sistema

Agrupen nodes conceptuals segons la seva funció dins el model d’aprenentatge.

Permeten entendre el sistema en capes.

Exemples:  
→ [[Arquitectura del sistema]]  
→ [[Dinàmiques del sistema]]  
→ [[Cognició en el sistema]]  
→ [[Límits del sistema]]

---

### 🔹 Espais d’anàlisi

Connecten els conceptes amb el desenvolupament de la recerca.

Exemples:  
→ [[Iteracions de la recerca]]  
→ [[Resultats]]  
→ [[Límits i tensions]]

Aquests espais mostren com el sistema evoluciona i quines evidències el sustenten.

---

### 🔹 Fonaments teòrics

Inclouen els marcs conceptuals que sustenten la investigació.

Exemples:  
→ [[Cognició distribuïda (marc teòric)]]  
→ [[Cognició de grup (marc teòric)]]  
→ [[Teoria Actor-Xarxa]]  
→ [[Aprenentatge col·laboratiu]]

Aquests elements permeten comprendre d’on provenen les idees del sistema viu.

---

### 🔹 Metadades i context acadèmic

Informació complementària que amplia la comprensió del treball.

Exemples:  
→ [[Anotacions de recerca]]  
→ [[Autors i referents]]  
→ [[Articles i llibres]]  
→ [[Recursos digitals]]

---

## Com interpretar aquests elements?

Cada element és una **porta d’entrada al coneixement**.

Alguns expliquen idees.  
Altres mostren processos.  
Altres connecten amb fonaments teòrics.  
Altres documenten l’evolució de la recerca.

Junts formen una xarxa interconnectada.
