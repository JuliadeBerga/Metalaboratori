# Dinàmica. Game Model

## Definició del Concepte
Les dinàmiques descriuen el comportament del jugador en relació amb el joc i amb els altres jugadors.

## Preguntes a respondre per definir la Dinàmica

La pregunta general és:  
Com es relacionen les persones que juguen?

D'aquesta poden derivar moltes preguntes...
Es jugarà de forma individual o col·lectiva?
El mode serà col·laboració o competició?
El meu joc serà igual que la resta de jugadors?
Tothom veu el mateix? 
Es poden fer aliances?
