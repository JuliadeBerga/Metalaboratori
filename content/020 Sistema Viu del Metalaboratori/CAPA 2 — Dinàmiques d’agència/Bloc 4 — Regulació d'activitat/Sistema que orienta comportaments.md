# Sistema que orienta comportaments

## Definició

Configuració en què el disseny del sistema influeix en les decisions i en les pràctiques dels participants.

## En el context del laboratori

En ellaboratori.cat, el sistema orienta el comportament dels participants a través de diversos mecanismes:

- fa visibles els assoliments dels equips
    
- estructura el progrés mitjançant el sistema de portes
    
- facilita l’accés a tutorials i recursos
    
- permet observar el treball d’altres equips
    

Aquests elements no obliguen directament a actuar d’una determinada manera, però **influeixen en les decisions dels participants**.

## Funcionament

Quan els estudiants observen:

- quines portes estan obertes
    
- quins projectes s’han desenvolupat
    
- quins recursos estan disponibles
    

aquesta informació orienta les seves decisions i les seves trajectòries dins el projecte.

## Funció dins la redistribució d’agència

El sistema no substitueix l’agència dels participants, però crea un entorn que **canalitza les seves accions**.

D’aquesta manera, l’activitat es regula parcialment a través del disseny de l’entorn.
