# Adaptació pedagògica

L’adaptació pedagògica descriu el procés mitjançant el qual el sistema del Metalaboratori ajusta progressivament les seves pràctiques educatives en resposta a les necessitats, dificultats i oportunitats que emergeixen durant el desenvolupament del projecte.

A mesura que el laboratori evoluciona al llarg de les diferents iteracions, els participants identifiquen situacions que requereixen modificacions en la manera com s’organitza l’activitat educativa.

Aquest procés permet que el sistema mantingui la seva coherència pedagògica mentre es transforma.

---

## Evidència empírica

Durant el desenvolupament del projecte es poden observar diverses adaptacions en el funcionament pedagògic del laboratori.

Entre aquestes destaquen:

- l’ajust de les dinàmiques de treball dels estudiants
- la reorganització dels dispositius d’avaluació
- la incorporació progressiva del sistema de portes
- l’aparició de noves formes d’orientació entre iguals
- la redefinició del rol del docent dins el sistema

Aquestes adaptacions apareixen com a resposta a les experiències acumulades durant les diferents iteracions del laboratori.

---

## Funcionament dins el sistema

L’adaptació pedagògica permet que el sistema d’aprenentatge es mantingui funcional en contextos canviants.

Quan una determinada configuració genera dificultats o oportunitats noves, el sistema pot ajustar diferents components del seu funcionament, incloent:

- l’organització de les activitats
- la distribució de responsabilitats entre participants
- els dispositius que regulen el progrés dels estudiants
- les pràctiques d’orientació dins la comunitat d’aprenentatge

Aquest mecanisme contribueix a la sostenibilitat del sistema al llarg del temps.

---

## Implicacions pedagògiques

L’adaptació pedagògica reforça la concepció del laboratori com un sistema d’aprenentatge viu.

En lloc de mantenir una estructura pedagògica rígida, el laboratori evoluciona a partir de l’experiència acumulada pels participants i de l’anàlisi de les dinàmiques que emergeixen durant el projecte.

Aquest procés permet que el sistema desenvolupi configuracions pedagògiques que responen millor a les necessitats del context educatiu.
