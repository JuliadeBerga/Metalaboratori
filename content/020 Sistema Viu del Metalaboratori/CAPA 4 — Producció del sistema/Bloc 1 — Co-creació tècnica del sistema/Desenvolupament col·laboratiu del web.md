# Desenvolupament col·laboratiu del web

El desenvolupament col·laboratiu del web descriu el procés mitjançant el qual estudiants i docents participen conjuntament en la construcció tècnica de l’entorn digital d’aprenentatge.

En el cas analitzat, ellaboratori.cat no és dissenyat íntegrament pel professorat ni implementat com una plataforma tancada. El sistema emergeix progressivament a través del treball coordinat d’un nucli d’estudiants que desenvolupen funcionalitats, integren components i resolen problemes tècnics en col·laboració amb el docent.

Aquest procés transforma el web en un artefacte sociotècnic híbrid: alhora producte pedagògic i infraestructura d’aprenentatge.

---

## Evidència empírica

Durant la Iteració 2, cinc estudiants participen directament en el desenvolupament del web durant les hores de treball associades a la Formació en Centres de Treball. 

Aquest nucli assumeix tasques com:

- desenvolupament de l’arquitectura MVC
- integració de bases de dades
- creació de noves funcionalitats
- resolució de problemes tècnics del sistema
- adaptació del web a les diferents fases del projecte educatiu

El treball es realitza en coordinació constant entre els membres del grup i amb el docent, generant un procés continu de discussió tècnica i presa de decisions.

El web no és només el resultat d’aquest procés, sinó també el seu suport: els components desenvolupats es validen directament dins el propi entorn digital.

---

## Mecanisme dins el sistema

Aquest procés produeix tres transformacions estructurals.

En primer lloc, el web deixa de ser un instrument extern per convertir-se en una infraestructura produïda pels participants del sistema.

En segon lloc, el desenvolupament tècnic esdevé una activitat cognitiva distribuïda, en què el coneixement es reparteix entre persones, codi, arquitectura i documentació.

Finalment, la participació en el desenvolupament del sistema proporciona als estudiants una comprensió estructural de l’entorn d’aprenentatge que posteriorment permet l’emergència de rols com el co-associat.
