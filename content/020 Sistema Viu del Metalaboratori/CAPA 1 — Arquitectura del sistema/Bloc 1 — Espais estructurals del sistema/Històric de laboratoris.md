# Històric de laboratoris

## Definició

Repositori de les diferents versions de ellaboratori.cat desenvolupades al llarg dels cursos acadèmics.

## En el sistema del laboratori

El web manté un registre de les versions anteriors del projecte, permetent observar l’evolució de la seva arquitectura, els gadgets desenvolupats i les decisions de disseny adoptades en cada iteració.

## Funció dins l’arquitectura

- preservar la memòria evolutiva del sistema
    
- documentar els processos de redisseny
    
- facilitar l’anàlisi retrospectiva del projecte
