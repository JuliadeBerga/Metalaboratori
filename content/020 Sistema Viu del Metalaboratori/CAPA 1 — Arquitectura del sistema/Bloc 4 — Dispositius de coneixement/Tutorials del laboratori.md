# Tutorials del laboratori

## Definició

Repositori que conserva les diferents versions de ellaboratori.cat desenvolupades al llarg del temps.

## En el sistema del laboratori

El web incorpora un accés a les versions anteriors del laboratori que permet observar com l’espai ha evolucionat amb els anys. Aquest registre mostra l’aparició de nous gadgets, modificacions d’interfície i canvis en l’organització del sistema.

## Funció dins l’arquitectura

- preservar la memòria evolutiva del sistema
    
- documentar els processos de redisseny
    
- permetre analitzar l’evolució del laboratori
