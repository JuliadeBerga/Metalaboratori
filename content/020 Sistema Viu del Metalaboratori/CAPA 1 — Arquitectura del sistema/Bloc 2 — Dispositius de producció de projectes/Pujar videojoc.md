# Pujar videojoc

## Definició

Funcionalitat del web que permet als equips DAM pujar versions del videojoc en desenvolupament.

## En el sistema del laboratori

Els estudiants poden actualitzar el joc a mesura que avança el projecte, permetent que altres participants (especialment els estudiants d’educació) el puguin veure i provar.

## Funció dins l’arquitectura

- permetre iteracions del projecte
    
- compartir versions del joc
    
- facilitar feedback entre col·lectius
