# Laable

## Definició

Buscador propi de ellaboratori.cat que permet localitzar persones, videojocs, tutorials i altres recursos generats dins el sistema.

## En el sistema del laboratori

Inspirat en el funcionament dels motors de cerca, Laable actua com una base de dades navegable que permet accedir als diferents elements produïts dins el laboratori.

## Funció dins l’arquitectura

- facilitar la navegació dins el sistema
    
- connectar persones, projectes i recursos
    
- estructurar la informació generada
