# Sistema de portes

## Definició

Dispositiu d’avaluació integrat a ellaboratori.cat que estructura el progrés dels estudiants mitjançant un conjunt de portes que es van obrint a mesura que s’assoleixen determinats objectius.

## En el sistema del laboratori

El sistema de portes funciona com un mecanisme de progressió visible que permet als estudiants avançar dins el projecte a partir dels assoliments aconseguits.

Cada porta representa un llindar d’aprenentatge o de desenvolupament del projecte. Quan un equip compleix les condicions associades, la porta s’obre i permet accedir al següent nivell d’activitat.

Aquest sistema es va implementar finalment com un **gadget independent dins del web**, convertint-se en un dels dispositius centrals de l’arquitectura del laboratori.

## Funció dins l’arquitectura

- estructurar la progressió del projecte
    
- integrar l’avaluació dins l’activitat
    
- fer visibles els assoliments dels equips
    
- permetre trajectòries diferenciades
    

## Característiques

El sistema de portes:

- no funciona com una qualificació final
    
- distribueix l’avaluació al llarg del procés
    
- converteix el progrés en un element visible del sistema
    
- permet distingir entre **mínims i màxims**
