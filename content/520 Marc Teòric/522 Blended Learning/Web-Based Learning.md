# Web-Based Learning

[[Oliver 2000]] utilitzen la teoría de l’aprenentatge situat per dividir en tres grans blocs la seva experiència educativa:

## · Activitats d’aprenentatge:

Col·laboració. Fomentant la col·laboració i la presa de decisions en grup.

Reflexió. L'avaluació entre iguals requereix processos reflexius 

Articulació. Les solucions resumides necessiten l'articulació i l'explicació de l'aprenentatge

## · Suports d’aprenentatge:

Coaching i Bastida. Els alumnes reben el suport d'altres membres del grup amb bastides.

Valoració Real.

Les estratègies d'avaluació avaluen tant els processos d'aprenentatge com els productes.

## · Continguts

Contextos i Activitats Reals.

Actuacions d'experts. Es proporcionen solucions de mostra per guiar els estudiants en els processos de resolució de problemes i Accés a llocs web d'experts i altres experimentats s'afegeix a les fonts d'informació

Múltiples perspectives:

• accés a diversos llocs web per obtenir informació

• una varietat de fonts de mitjans, p. impressió, vídeo

• els agrupaments proporcionen diferents perspectives

• Les millors solucions presenten perspectives alternatives
